Include an HTML file issued from a R Markdown file

Open in a new tab

The follow picture is clickable, to open the HTML file in a browser tab:

The code is:

<a href="/html/just_a_page.html">
  <img src="/image/open_in_new_tab.jpg" alt="." width="30vw">
</a>

Embed the HTML page

The code is:

<iframe 
  width="100%"
  height="2300"
  src="/html/just_a_page.html"
  frameborder="0"
  allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>
